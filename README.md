## Closure

Teniendo dos funciones **outer function** e **inner function** el concepto de closure es el siguiente:

* _La **inner function** esta declarada dentro de la **outer function** y la **inner function** sólo está disponible dentro del cuerpo de la **outer function**_

* _La **inner function** tiene acceso a las variables de la **outer function** donde fué declarada_
